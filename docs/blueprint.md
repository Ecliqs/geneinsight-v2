# **App Name**: GeneInsight Connect

## Core Features:

- Hero Section Display: Displays the main value proposition, primary headlines, calls-to-action (CTAs), and compelling medical visuals.
- Dynamic Content Rendering: Renders all informational sections (e.g., Proof Strip, Problem, Solution, Capabilities, Trust, How It Works, Quality, When to Consider) using specified layouts like card-based, grid, checklist, and horizontal steps for maximum scannability and visual appeal.
- Persistent Call-to-Action System: Implements sticky and floating 'Discuss a Case' buttons that link directly to Calendly, ensuring prominent visibility and easy access for users across all devices.
- Direct Contact Integration: Provides quick access to additional contact options via dedicated click-to-call and WhatsApp chat buttons, especially visible and accessible on mobile devices.
- Calendly Booking Widget: Embeds and displays the Calendly scheduling interface directly within the landing page for seamless booking of 30-minute case discussions.
- Adaptive Responsive Design: Ensures the entire landing page is fully responsive across desktop, tablet, and mobile, adhering to a mobile-first design philosophy with clean content stacking and key CTAs visible above the fold on mobile.
- AI-Powered Genomic Interpretation: Utilizes advanced AI tools to analyze multi-omics data, providing sophisticated insights and interpretations for complex genomic diagnostics, thereby aiding clinicians in identifying root causes.

## Style Guidelines:

- Primary color: A vibrant, clinical blue (#258EDB) to convey trust and precision, drawing inspiration from the GeneInsight logo. It offers a strong contrast against lighter backgrounds.
- Background color: A very light, cool off-white (#F5F8FA) that is subtly tinted with the primary hue, providing a clean and expansive canvas for content and strong whitespace.
- Accent color: A sophisticated, slightly darker blue-violet (#3359CC) to provide contrast and visual interest in interactive elements without detracting from the clinical aesthetic.
- Body and headline font: 'Inter', a grotesque-style sans-serif. Its modern, objective, and neutral look supports the clinical and premium theme, ensuring high legibility and a clean aesthetic for all content.
- Utilize modern, clean line icons and high-quality, scientifically accurate medical illustrations, focusing on themes like DNA, genomics, lab visuals, and microscopy/cellular imagery.
- A mobile-first responsive design featuring extensive whitespace, card-based and grid layouts for scannable content, with CTAs prominently visible above the fold on mobile and sections stacking cleanly.
- Subtle, smooth animations for interactive elements and section transitions on scroll, contributing to a premium feel without being flashy or distracting.
